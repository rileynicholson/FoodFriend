# FoodFriend!

FoodFriend! is a Java program that helps a user come up with what to make with the ingredients they currently have in their pantry! FoodFriend gives ideas ranging from something as small as a snack to as big as a regular course meal. This project is in early development, so a lot of the features this program offers are still being developed!
